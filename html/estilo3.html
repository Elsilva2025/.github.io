ESTILO LISO 3

1. Primero utiliza un cortauñas para darle forma a la uña
2. Luego utiliza una lima para limar la uña
3. Luego usa una maquina para limar la cuticula
4. Despues usa una brocha para retirar el polvo de las uñas
5. Despues con el esmalte rosa pinta la uña y con uno blanso hace diseños de corazon
6. Por ultimo sella con un esmalte transparente
